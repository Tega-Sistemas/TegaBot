Termos de Serviço (ToS)

Bem-vindo ao TegaBot. Ao convidar ou utilizar este bot em seu servidor Discord, você concorda com os seguintes termos:

1. Uso do Bot

O TegaBot é uma ferramenta corporativa privada destinada a integrar notificações de sistema e consultas de suporte via API interna. O uso é restrito a servidores e usuários autorizados pela Tega Sistemas.

2. Disponibilidade

Este serviço é fornecido "como está", sem garantias de tempo de atividade (uptime) ou funcionalidade contínua. Podemos interromper o serviço para manutenção ou atualizações sem aviso prévio.

3. Responsabilidade

Não nos responsabilizamos por quaisquer danos diretos ou indiretos decorrentes do uso ou incapacidade de usar o bot.

Política de Privacidade

A sua privacidade é importante para nós. Esta política descreve como o TegaBot coleta e processa dados.

1. Dados Coletados

O TegaBot pode coletar e processar as seguintes informações limitadas:

  • IDs de Usuário e Servidor (Guild): Utilizados apenas para funcionamento técnico e logs operacionais.
  • Conteúdo de Mensagens: O bot processa mensagens que iniciam com o prefixo configurado (ex: !chamado) para executar comandos. O conteúdo dessas mensagens pode ser enviado para nossa API interna para processamento da solicitação.
  • Logs Técnicos: Registros temporários de erros e execução para fins de depuração.

2. Como Usamos os Dados

Os dados são utilizados exclusivamente para:

  • Responder aos comandos solicitados pelo usuário (ex: consulta de chamados de suporte).
  • Notificar canais configurados sobre eventos recebidos via Webhook.

3. Compartilhamento de Dados

Não compartilhamos, vendemos ou alugamos dados de usuários a terceiros. Os dados trafegam apenas entre o Discord e nossa infraestrutura interna.

4. Contato

Para dúvidas sobre estes termos ou sobre o bot, entre em contato com a equipe de suporte da Tega Sistemas.
